# bike-sales-dashboard-excel

A comprehensive dataset and pre-calculated pivot summaries designed for building an Interactive Bike Sales Dashboard. This dataset enables analysis of customer demographics, financial attributes, and commuting behavior to uncover factors influencing bicycle purchases.
<br>
## Project Overview

The Bike Sales Dashboard Dataset contains customer-level data along with summarized pivot tables that support data visualization and business intelligence reporting.

The dataset can be used to:

- Analyze customer purchasing behavior.
- Identify demographic trends related to bike purchases.
- Explore relationships between income, commute distance, and buying decisions.

Using this dataset, you can explore questions such as:
